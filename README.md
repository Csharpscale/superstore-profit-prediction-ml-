📊 Superstore Profit Prediction (Machine Learning)

📌 Overview

This project focuses on predicting Profit using the Superstore dataset.
It demonstrates a complete Machine Learning workflow including data preprocessing, visualization, feature engineering, and model evaluation.

---

🎯 Objectives

- Clean and preprocess the dataset
- Perform exploratory data analysis (EDA)
- Visualize relationships between variables
- Build a Linear Regression model
- Evaluate model performance

---

🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

📈 Key Insights

- Most sales occur in the lower range (0–5000)
- Profit shows a weak positive relationship with sales
- Presence of outliers (high profit and high loss)
- Data shows weak linear correlation

---

🤖 Model Used

- Linear Regression

---

📊 Model Performance

- R² Score: Negative
- Adjusted R²: Negative

➡️ This indicates that Linear Regression is not a suitable model for this dataset.

---

🚀 Future Improvements

- Try advanced models like Decision Tree and Random Forest
- Improve feature engineering
- Handle outliers more effectively
- Apply scaling and normalization

---

📁 Dataset

- Superstore dataset (CSV format)

---

✅ Conclusion

This project demonstrates the end-to-end Machine Learning pipeline.
Although the model performance was poor, it provides valuable insights into data behavior and model limitations.
